Colour game guide

The point of this game is to train the player to read RGB code correctly.

Rgb is written as: RGB(Red, Green, Blue).

The higher the number between the brackets the hue of that same number will be bolder (brighter).

How the game works:

◌ Read the RGB code written on the screen.

◌ Press the circle you think holds that colour.

◌ If you pressed the correct answer it will colour the others the same.

◌ Else the circle you pressed will vanish from the screen.

◌ To start another round or get different colours press the "RESET" button.
